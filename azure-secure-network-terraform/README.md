Project overview:
# Azure Secure Network Foundation

## Overview
This project creates a secure Azure network foundation using Terraform. It includes:

- Resource Group
- Virtual Network (VNet)
- Subnets (Web, App, DB)
- Network Security Groups (NSG)
- NSG Rules (Allow HTTP)
